# CODSOFT Projects

A collection of Python projects developed as part of the CODSOFT internship program.

## Projects

### 1. AI Chatbot
A rule-based conversational chatbot built with Python.

**Features:**
- Greeting responses
- Farewell messages
- Help information
- Weather queries
- Basic conversation handling

**Run:**
```bash
python "ai chatbot.py"
```

---

### 2. Movie Recommendation System
A simple movie recommendation tool based on genre preferences.

**Features:**
- Multiple genre categories (Action, Comedy, Drama, Horror, etc.)
- Random movie suggestions from selected genre
- Interactive console interface

**Run:**
```bash
python "Movie recommendation system.py"
```

---

### 3. Tic-Tac-Toe
A GUI-based Tic-Tac-Toe game using Python's `tkinter` library.

**Features:**
- Graphical user interface
- Player vs Computer gameplay
- Win/draw detection
- Clean visual design

**Run:**
```bash
python "Tic-tac-toe.py"
```

---

## Requirements

- Python 3.x
- No external dependencies required (uses built-in libraries)
